********************* Contexte ************************
Réalisation d'une application Java JSF 

Délai : Application à réaliser pour le 31/10/2019 17h30.

Description : Adaming Park est une application pour la gestion d'un parc d'attraction.
	      Elle permet aux visiteurs de visualiser les attractions du parc et de commander leur billets.
	      Elle permet aux admin de s'authentifier et gérer les différentes pages du site.

	3. Difficultés rencontrées et solutions apportées
La principale difficulté que nous avons rencontrée a résidé dans l'évaluation du temps nécessaire pour la réalisation de chaque partie.
En effet, l'estimation du temps de travail nécessaire pour la partie front et back a été un peu erronée avec une légère surestimation du temps nécessaire pour effectuer le backend.
Une autre difficulté que nous avons rencontré est dans l'attribution des noms des différents fichiers : un joli mélange de minucscules, majuscules, français et anglais a été effectué. Ce sera un point auquel il faudra faire attention pour les prochains projets
Une erreur dans la conception a été mise en évidence lors de la fin du projet concernant les types de billets qu'il est possible de commander. En effet, il faut rentrer manuellement dans la base de données via l'invite de commande le type normal et réduit ainsi que leur prix associé.  


	4. Conclusion

Améliorations à faire : 
	- rendre dynamique les types de billets pour ne plus avoir à passer par l'invit de commande.
